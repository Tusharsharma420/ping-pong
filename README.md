# Ping Pong

A lightweight browser ping pong game built with HTML, CSS, and JavaScript.

## Play

Open `index.html` in a browser, or serve the folder locally.

## Controls

- `Space` or click: serve
- `W` / `S` or arrow keys: move paddle
- Mouse: move paddle
- `P`: pause or resume
- `R`: restart
- `F`: fullscreen

First side to 7 points wins.
